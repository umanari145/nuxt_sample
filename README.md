# nuxt sample


## プロジェクトの作り方


```
 npm install -g @vue/cli-init
```

nuxt_sampleというプロジェクトを作り方
```
vue init nuxt-community/starter-template nuxt_sample
#ウィザードに答えてプロジェクトを作成
? Project name nuxt_sample
? Project description Nuxt.js project
? Author umanari145 <umanari145@gmail.com>
```
